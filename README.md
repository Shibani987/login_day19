# Animated Login and Signup Pages

Responsive animated login and signup pages built with HTML, CSS, and JavaScript.

## Features

- Split-screen modern login layout
- Matching signup page
- Animated background elements
- Responsive mobile layout
- Password show/hide button
- Basic form validation
- Success message on submit

## How To Run

Open `index.html` directly in your browser for login.

Open `signup.html` directly in your browser for signup.

No installation or server is required.
